Typescript
----------
Typescript feature app.

Install
-------
1. npm install
>See **package.json** for details.

Update
------
1. npm install -g npm-check-updates
2. ncu
3. ncu -u

Build
-----
>Runs compile and lint.
1. npm run build

App
---
>Runs build script and node index.
1. npm run app
